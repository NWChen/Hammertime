## HammerTime Web App

### Install
1. `git clone https://github.com/NWChen/Hammertime`
2. `cd web`
3. `pip install -r requirements.txt`
4. `python app.py`

### Deploy
- `cd Hammertime`
- `git subtree push --prefix web heroku master`
